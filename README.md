# Phased App Builder

An experiment in building apps through structured prompts with clear phase gates.

## Concept

Build apps in distinct phases, each with defined benchmarks before proceeding.

## Phases (Draft)

| Phase | Focus | Exit Criteria |
|-------|-------|---------------|
| 1. Architecture | Stack, structure, data model, constraints | Documented decisions |
| 2. Design | Components, screens, user flows | Wireframes or specs |
| 3. Implementation | Build in increments | Working code per increment |
| 4. Validation | Tests, review | Benchmarks met |
